# RDOS v22 — 상세 문헌연구 엔진 (수정안 반영)

## 문헌연구 (`/literature-review`)

- **통합 검색 소스**: OpenAlex, Crossref, Semantic Scholar에 DOAJ, arXiv, PubMed 추가 (해외 검색).
- **검색 랭킹**: 최신성·인용·프로젝트 연구유형(양적/질적/혼합/실험)·키워드 적합도 반영.
- **데이터분석 탭**: 연도·저널·소스 분포 차트, 막대/선/Lollipop/도넛 유형 선택.
- **연구 인사이트 탭**: 구조화 요약, 연구방법 요약, 논문 비교, AI 연구 비평, 문헌고찰 초안, Research Card.
- **네트워크분석**: 노드 크기 축소, 그래프 시각 스타일(기본 / Pictogram / Isotype / Infographic).
- **Google Material Symbols** (Apache-2.0) — 차트·그래프 UI 아이콘.
