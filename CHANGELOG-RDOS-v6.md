# CHANGELOG — AI Research OS + RDOS v6

v5 기반. 슈퍼관리자 노출 정밀화·RDOS 관리자 연동·논문구조엔진 UI·RDOS 학습
콘텐츠 대폭 증설·이메일 마스킹·RDOS 사이드바 가독성을 해결. 버전 6.0.0.

## 1. RDOS 관리자 — 슈퍼관리자 전용 노출 + 3개 탭 실연동
- RDOS 관리자 페이지/사이드바 링크는 슈퍼관리자(내장 기본 계정)에게만 노출(기존 유지·확인).
- **연동 문제 해결**: 승인 대기/회원/메뉴 관리 세 탭이 각각 실제 데이터와 동작.
  - `/api/admin/rdos/enrollments` 를 **profiles(전체 회원 명부) 병합**으로 확장 →
    회원 탭에서 일반 회원에게도 RDOS 접근을 직접 부여/차단/퇴출 가능.
  - `/api/admin/rdos/approve` 를 update→**upsert** 로 변경(자가 가입 행이 없어도 부여 가능).
  - 관리자 뷰를 탭별 독립 로딩·상태 표시·오류 가시화로 재작성.
- RDOS 사이드바 헤더 둘째 줄: `연구준비자` → **`Researcher Development Operating System`**.

## 2. AI-Research-OS 논문구조엔진 UI
- 우측 Copilot 패널에 있던 **자동 저장됨 · 상세 에디터** 를 상단 브레드크럼(상단 메뉴)으로
  이동하고 **한 줄**로 정리(두 줄 → 한 줄). 패널 헤더는 장수 표시만 남김.

## 3. RDOS 본페이지 — 논문가이드 5-4 전체 적용·대폭 증설
- 각 메뉴 학습목표 3개 → **5~6개**로 확장.
- 학습 모듈(레슨) **24개 → 56개**(퀴즈 112문항)로 증설. 모두 본문 HTML + 이해도 퀴즈.
  - 출처: 논문가이드 8개 장(서술구조·탈락패턴 15·좋은문장 15) + 31개 연구용어
    + 양적 9 / 질적 7 / 혼합 4 방법론 카탈로그.

  (연구입문~논문심사)을 노출.

## 4. 슈퍼관리자 이메일 마스킹(실행 무중단)
- 내장 슈퍼관리자 이메일을 소스에 평문 대신 **base64 로 가림**(atob 동기 디코드).
  → 코드/문서 어디에도 평문 이메일이 노출되지 않으며 판정 로직은 그대로 동작.
- 환경변수(`SUPER_ADMIN_EMAILS` / `NEXT_PUBLIC_SUPER_ADMIN_EMAILS`)로 추가/회전 가능.
