## 1. DesignMe 스타일 변환 (랜딩 + 전체메뉴/기능 Framer Motion 데모)

designme.agency의 디자인 언어를 분석하여 다음 요소를 적용했습니다.

- **시그니처 스크램블 텍스트**: 글자가 무작위로 셔플되며 원문으로 안착하는 효과.
  새 컴포넌트 `components/marketing/scramble-text.tsx` (viewport 진입 / hover / mount 트리거,
  `prefers-reduced-motion` 존중). 랜딩 헤드라인과 히어로 CTA 버튼(hover)에 적용.
- **번호 매긴 리스트(01, 02 …)**: 5개 메뉴 흐름 쇼케이스를 좌측 번호 리스트 + 우측 데모
  2열 레이아웃으로 재구성(DesignMe 서비스 섹션 스타일). 활성 항목 레일(`layoutId`) 슬라이드.
- **모노스페이스 메타 라벨**: 소문자 섹션 라벨("menu flows in motion" 등), 데모 프레임 상단의
  브라우저 크롬(신호등 + `ai-research-os / {menu}` + `live demo`) 등 모노 메타 텍스트.
- **큰 디스플레이 타이포 + 좌측 정렬**: 명조 헤드라인을 `clamp(2rem,5.5vw,4rem)`로 확대,
  중앙 정렬 → 좌측 정렬 편집형 구성. 라임(#c8ff4d) 액센트 도트.
- **전체 메뉴 그리드**: 카드에 인덱스 번호(우상단), hover 시 언더라인 확장·아이콘 회전 등
  마이크로 인터랙션 추가.
- 적용 파일: `components/marketing/motion-showcase.tsx`(AllFeaturesMotion·PaperAnalysisFlow),
  `components/marketing/menu-flow-showcase.tsx`(5개 메뉴 흐름), `app/page.tsx`(히어로).

각 데모의 **모션 동작(업로드→분석→결과 stagger 등)은 그대로 유지**하고 외형만 DesignMe 톤으로
변환했습니다(다크 테마 위에 따뜻한 미니멀·편집형 레이아웃·모노 디테일).

## 2. 논문 크리틱 메뉴 수정 (첨부 이미지 기준)

- **버튼명 변경**: "논문 구조" → **"논문 불러오기"** (탭·패널 헤더 공통, ko/en/zh 모두).
  `paperStructure` 키: 논문 불러오기 / Load paper / 载入论文.
- **변환텍스트 크리틱 번호 미생성 문제 수정**:
  - 원인 — 원문 이미지 번호는 `c.pageNum === pg.pageNum` 기준인데, 변환텍스트 번호는
    별도의 텍스트 매칭(relatedCards)만 사용해, 페이지 단위 크리틱(본문 텍스트가 빈 카드)이
    변환텍스트에서 누락되었음.
  - 수정 — 변환텍스트도 **원문 이미지와 동일한 "같은 페이지의 모든 카드" 기준**으로 통일하고,
    텍스트 매칭 카드와 합집합으로 표시. 페이지 헤더 카운트도 동일 기준으로 일치.
  - 추가 개선 — 변환텍스트를 **문장 단위로 분할**하여 매칭되는 문장 옆에 인라인 번호 배지를
    배치(해당 문장 하이라이트), 문장 매칭이 안 되는 카드는 페이지 상단에 모아 표시.
    번호 ↔ 패널 카드 양방향 연동/강조 유지.
