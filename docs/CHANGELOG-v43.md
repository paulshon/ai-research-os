# CHANGELOG — v43

첨부된 모바일/데스크탑 피드백 3건과 "모바일 전체 글자 축소" 요청을 반영한 릴리스입니다.

---

## 1. 모바일(≤767px) 전체 글자 −1.5pt 축소

- `globals.css`에 모바일 전용 `@media (max-width: 767px)` 오버라이드를 추가하여, 사용 중인
  모든 글자 크기 토큰(`text-[Npx]`)을 약 −2px(≈1.5pt) 축소.
- 데스크탑/태블릿(≥768px)은 영향 없음. 12px 미만의 보조 라벨은 가독성을 위해 유지.
- 랜딩 히어로 헤드라인은 인라인 clamp라 이 규칙의 영향을 받지 않으며, 아래 3번에서 별도 +2pt 처리.

## 2. 사이드바 — Research Flow / AI Tools 아코디언 (image 1)

- 두 섹션 헤더를 **접기/펼치기 토글(＋/－)** 버튼으로 변경.
- 모바일에서 사이드바가 길어 AI Tools 하위 메뉴가 가려지던 문제 해소
  (Research Flow를 접으면 AI Tools 항목이 바로 보임). 기본값은 펼침.
- 데스크탑/모바일 공통 사이드바에 반영.

## 3. 랜딩(모바일 첫 화면) (image 2)

- 헤더와 본문 사이의 **과도한 상단 빈 공백 제거**: 히어로 상단 여백을 모바일에서 축소
  (`pt-[150px]`/`pt-10` → 모바일 `pt-[84px]`/`pt-2`, 데스크탑 값 유지).
- 모바일 히어로 헤드라인("학술적 AI 알고리즘과 …운영체계") **+2pt 확대**
  (clamp 최소값 1.75rem → 1.92rem, 데스크탑 최대값은 그대로).

## 4. 논문작성 — 그림/표 가져오기 + PDF 저장 (image 3)

### 4-1. 가져오기(불러오기)
- 상단 툴바에 **「가져오기」** 버튼 추가. 클릭 시 OS 파일 선택창이 열려 디렉토리 이동 후 파일 선택.
- 지원 포맷:
  - 이미지(JPEG·PNG·GIF·WEBP·BMP·SVG) → 본문에 `<img>`로 삽입.
  - CSV → 표로 파싱하여 삽입.
  - Excel(XLSX·XLS) → 첫 시트를 표로 변환하여 삽입(SheetJS `xlsx` 사용, 동적 import).
  - 기타 텍스트 → 단락으로 삽입.
- 에디터 핸들에 `insertHTML`를 추가하여 이미지/표 HTML 삽입을 지원.
- 처리 상태(삽입 완료/실패) 안내 바 표시.

### 4-2. PDF 저장
- 상단의 저장 버튼을 **「PDF 저장」** 으로 변경. 클릭 시 인쇄 대화상자가 열려
  **‘PDF로 저장’ 선택 + 디렉토리·파일명 지정**이 가능(브라우저 기본 PDF 저장 경로).
- 현재 섹션 제목·프로젝트명·본문(이미지/표 포함)을 학술 서식(세리프, 표/이미지 스타일)으로 인쇄.
- 추가 라이브러리 없이 동작(window.print 기반).
