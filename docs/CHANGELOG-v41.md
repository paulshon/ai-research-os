## 1. 문헌연구 — "네트워크분석" 탭 신설

문헌연구(`/literature-review`) 상단 탭에 **네트워크분석**을 추가했습니다.
검색으로 수집한 논문 메타데이터(저자·키워드·저널)를 기반으로 계량서지학(bibliometric)
네트워크를 생성합니다.

- 공동저자 네트워크 (Co-authorship): 같은 논문을 공동집필한 저자 ↔ 저자
- 키워드 동시출현 네트워크 (Co-word): 같은 논문에 함께 등장한 키워드 ↔ 키워드
- 저자-키워드 네트워크 (Author–Keyword)
- 저널 결합 (Source coupling)

외부 차트 라이브러리 없이 경량 force-directed 레이아웃을 자체 계산하여 SVG로 렌더하며,
연결 정도(degree) 중심성·연결요소(군집)·최소연결 필터를 제공합니다.

- 신규: `apps/web/components/literature/network-analysis.tsx`
- 분석 대상은 "선택한 논문 우선, 없으면 현재 지역 검색 결과 전체".

## 2. 문헌연구 — "파이프라인" 탭 제거 + 분석 버튼 ↔ 탭 연동 + 버튼 재디자인

- 사용 빈도가 낮던 **파이프라인(아키텍처 설명) 탭을 제거**하고 그 자리에 네트워크분석을 배치.
- 우측 플로팅 버튼(연구갭/연구설계/군집분석)과 상단 동명 탭을 **하나의 동작으로 연동**.
  버튼을 누르면 분석을 실행하고 해당 탭으로 이동하며, 탭을 눌러도 동일하게 동작합니다.
- 화려한 3색 그라데이션 버튼을 **글래스(backdrop-blur) 단일 패널**로 재디자인.
  선택 논문 수, 각 분석의 완료 상태(점 표시)를 한눈에 보여줍니다.

## 3. 해외 검색 최대 추출량 확대

해외(international) 검색 시 결과 수 상한을 **50 → 200**으로 올리고, 소스별 호출량을
각 API 한계까지 끌어올렸습니다(OpenAlex ≤200, CrossRef ≤200, Semantic Scholar ≤100).

- `apps/web/app/api/scholar/route.ts`: limit 상한 200, international 분기 소스별 최대치 호출.
- 프런트(`literature-review`)는 해외 검색에 한해 100건을 요청하도록 변경.

## 4. "워크플로우" + "제출 준비" → "논문일정"으로 통합

검토·검증 흐름의 두 메뉴를 하나의 **논문일정(`/schedule`)** 페이지로 통합했습니다.

- 상단: 8단계 연구 일정·진행률·태스크 관리(구 워크플로우)
- 하단: 제출 전 최종 점검 체크리스트 + 원고 내보내기(구 제출 준비)
- 신규: `apps/web/app/(dashboard)/schedule/page.tsx`
- 구 경로 `/workflow`, `/submission`은 `/schedule`로 자동 리다이렉트.
- 사이드바 Research Flow에 "논문일정"(달력 아이콘)으로 배치, i18n(ko/en/zh) 키 추가.

## 5. 검토·검증 — "검토·검증 센터" 재구축 (표절/맞춤법/AI 생성 탐지 + 통합품질보고서)

검증(`/validation`)을 IA 설계에 맞춰 **문장 단위(Sentence Map) 공유 엔진** 기반으로 재구축.
하나의 본문을 입력하면 "페이지 → 문단 → 문장" 좌표를 구축하고, 표절·맞춤법·AI 탐지가
**동일한 문장 좌표를 공유**합니다.

탭 구성: 표절검사 / 맞춤법검사 / AI 생성 탐지 / AI 심층검증(Gemini) / 통합품질보고서

- 표절검사: 문서 내부 중복(자기표절) n-gram 유사도 + 인용부호-출처 누락 탐지, 페이지별 막대.
- 맞춤법검사: 띄어쓰기/문장부호/맞춤법 쌍/학술문체/문장 길이 휴리스틱, 정확도 점수.
- AI 생성 탐지: perplexity 근사·burstiness·엔트로피·반복도로 문장별 AI 확률 산출.
- AI 심층검증: 기존 Gemini 검증(APA/Gap/Effect Size/논리/구조) 모드 유지.
- 통합품질보고서: 세 지표를 가중 합산하여 A+~D 등급과 수정 우선순위 제시.
- 신규: `apps/web/lib/validation/sentence-engine.ts`
- 모든 로컬 분석은 API 키 없이 항상 동작(결정적 알고리즘).

## 6. "AI 멘토링" + "AI 채팅" 메뉴 제거

- 사이드바/태블릿 레일/모바일 내비에서 **AI 멘토링(advisor)·AI 채팅(chat)** 제거.

## 7. 참고문헌 — 외부 서지정보 파일 불러오기 정상화

`참고문헌 정리`에서 RIS/BibTeX/EndNote(.enw)/PubMed(.nbib) **파일 업로드가 무시되던 문제**와
**붙여넣기가 1건만 인식되던 문제**를 수정했습니다.

- 신규: `apps/web/lib/citation/bib-file-import.ts`
  - 서지 텍스트 포맷(.ris/.bib/.bibtex/.nbib/.enw/.txt)과 문서(.pdf/.docx/.hwp 등)를 분리 처리.
  - RIS는 `ER` 단위, BibTeX는 `@entry` 단위로 **여러 건 동시 파싱**.
  - EndNote(.enw)·PubMed(.nbib) 전용 파서 추가.
- 파일 선택 `accept`에 서지 포맷 확장자 추가, 다중 건 임포트 토스트 안내.
