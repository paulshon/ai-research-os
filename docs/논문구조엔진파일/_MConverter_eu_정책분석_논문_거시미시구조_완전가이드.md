| **제1부: 거시적 구조 --- Bardach의 Eightfold Path 기반 각 장 간 연결 논리** |

Bardach(2012)는 정책 분석을 8단계(Eightfold Path)로 체계화한다. 이 8단계는 정책 분석 논문의 거시적 구조---즉 각 장이 어떤 역할을 수행하고 장과 장이 어떤 논리로 연결되는가---의 핵심 골격이다. 각 단계는 독립된 장이 아니라 논리적 사슬을 형성하며, 이전 단계의 산출물이 다음 단계의 논리적 전제가 된다.

|  |  |  |  |
|:--:|:--:|:--:|:--:|
| **장** | **핵심 질문** | **핵심 산출물** | **다음 장으로의 연결** |
| 1\. 서론/문제 정의 | 왜 이 정책 문제인가? | 문제 진술 + 분석 범위 | 문제 → 증거 수집 방향 |
| 2\. 증거 수집/현황 분석 | 문제의 실태는 어떠한가? | 실증적 현황 데이터 | 현황 → 대안 설계 근거 |
| 3\. 이론적 배경/분석틀 | 어떤 렌즈로 분석하는가? | 분석 프레임워크 | 분석틀 → 기준·대안 도출 |
| 4\. 정책 대안 구성 | 어떤 대안이 가능한가? | 2\~5개 정책 대안 | 대안 → 평가 기준 적용 |
| 5\. 평가 기준 선정 | 무엇으로 비교하는가? | 효과성·효율성·형평성 등 | 기준 → 대안 비교 평가 |
| 6\. 대안 평가/비교 분석 | 각 대안의 성과는? | 대안별 기준별 평가 행렬 | 평가 → 최적안 도출 |
| 7\. 논의/상충 분석 | 상충은 어떻게 해결하나? | 상충 해소 전략 + 이론 기여 | 상충 → 최종 권고 |
| 8\. 결론/권고·실행 전략 | 무엇을 제안하는가? | 최적안 + 실행 로드맵 | 수미상관: 문제 해결 |

**제1장. 서론 / 문제 정의 (Problem Definition)**

**1.1 거시적 기능**

정책 분석 논문의 서론은 Bardach(2012)의 제1단계 \'문제 정의(Define the Problem)\'에 대응한다. 이 장은 단순한 배경 서술이 아니라, 왜 이 문제가 정책적 개입을 요구하는가를 논증하는 정책 문제의 공식적 구성(problem construction) 과정이다. Stone(2012)은 모든 정책 분석은 문제 정의에서 시작되며, 문제를 어떻게 정의하느냐가 이후 모든 대안·기준·평가를 결정한다고 강조한다.

<p>① 정책 문제의 사회적·제도적 맥락 제기: 문제가 발생하는 거시적 환경과 제도적 조건</p>

**제2장. 증거 수집 / 현황 분석 (Assembling Evidence)**

**2.1 거시적 기능**

Bardach(2012)의 제2단계 \'증거 수집(Assemble Some Evidence)\'에 대응한다. 이 장은 정책 문제의 실태를 실증적으로 규명하는 단계로, 문제의 규모·추이·분포·영향을 데이터에 기반하여 체계적으로 제시한다. 단순 현황 나열이 아니라, 제1장에서 구성한 인과 이야기를 실증적으로 검증·보강하는 기능을 수행한다.

**제3장. 이론적 배경 / 분석 프레임워크 (Analytical Framework)**

**3.1 거시적 기능**

정책 분석 논문의 이론적 배경은 순수 학술 논문과 달리, 정책 문제의 인과 구조를 설명하고 대안 설계의 논리적 근거를 제공하는 \'분석 프레임워크\' 구축에 초점을 맞춘다. Stone(2012)은 정책 분석에서 사용되는 이론이 중립적 도구가 아니라 특정 가치와 세계관을 내포한다고 경고하며, 분석틀 선택의 규범적 함의를 명시할 것을 요구한다.

<p>Stage 1: 정책 개념 문제 제기 — 핵심 정책 개념(형평성·효율성·자유·안전 등)의 다의성과 정치적 논쟁성 확인</p>
<p>Stage 2: 핵심 정책 개념 정의 — Stone의 '정의의 정치(politics of definition)' 관점에서 개념 정의의 선택과 정당화</p>
<p>Stage 3: 주요 정책 이론 체계 정리 — 시장 실패 이론, 정부 실패 이론, 제도주의, 행동경제학 등 대표 이론의 핵심 명제</p>
<p>Stage 4: 기존 정책 연구 동향 분석 — 이 정책 영역의 연구 계보와 패러다임 전환</p>
<p>Stage 5: 이론 간 비교·비판 — 각 이론의 설명력·한계·정책 처방 방향의 차이 비교</p>
<p>Stage 6: 분석틀 통합 구성 — 복수 이론을 통합하여 이 연구의 분석 프레임워크 구축</p>
<p>Stage 7: 평가 기준 도출 — 분석틀에서 정책 대안을 비교할 평가 기준(criteria) 체계 도출</p></td>

**제4장. 정책 대안 구성 (Construct the Alternatives)**

**4.1 거시적 기능**

Bardach(2012)의 제3단계 \'대안 구성(Construct the Alternatives)\'에 대응한다. 이 장은 정책 분석의 핵심 창조적 단계로, 문제 해결을 위한 2\~5개의 실행 가능한 정책 대안을 체계적으로 설계한다. 대안 구성은 단순한 아이디어 나열이 아니라, 문제의 인과 구조에 기반하여 개입 지점(leverage point)을 식별하고, 각 대안이 어떤 인과 경로를 통해 문제를 해결하려는지를 논리적으로 설명하는 작업이다.

**제5장. 평가 기준 선정 (Select the Criteria)**

**5.1 거시적 기능**

Bardach(2012)의 제4단계 \'기준 선정(Select the Criteria)\'에 대응한다. 평가 기준은 정책 대안을 비교·평가하는 잣대로, 정책 분석의 규범적 핵심이다. Stone(2012)은 모든 평가 기준이 가치 판단을 내포하며, 기준 선택 자체가 정치적 행위임을 강조한다. 따라서 기준 선정의 근거와 기준 간 우선순위의 정당화가 필수적이다.

<p>① 효과성(Effectiveness): 정책 목표 달성 정도</p>

**제6장. 대안 평가 / 비교 분석 (Project the Outcomes & Confront Trade-offs)**

**6.1 거시적 기능**

Bardach(2012)의 제5·6단계 \'결과 예측(Project the Outcomes)\'과 \'상충 직면(Confront the Trade-offs)\'에 대응한다. 이 장은 각 정책 대안을 제5장에서 선정한 평가 기준에 따라 체계적으로 비교·평가하는 정책 분석의 분석적 핵심부이다. 평가 행렬(evaluation matrix)을 구성하여 각 대안의 강점과 약점을 가시화하고, 대안 간 상충(trade-offs)을 명시적으로 다룬다.

**제7장. 논의 / 상충 분석 (Discussion)**

**7.1 거시적 기능**

논의 장은 대안 평가 결과의 의미를 이론과 연결하여 학문적 기여를 확립하고, 정책적 함의를 도출하는 장이다. Stone(2012)이 강조하는 정책 역설(policy paradox)---효율성과 형평성의 상충, 자유와 안전의 긴장---이 이 장에서 본격적으로 다루어진다. 심사위원이 논의장에서 묻는 핵심 질문은: \'So what? 이 분석이 정책 결정에 왜 중요한가?\'이다.

**정책 문제 정의의 5문단 미시 서술 구조**

정책 문제 정의는 학술 논문의 연구배경과 유사하되, Stone(2012)이 강조하는 인과 이야기(causal story)의 구성과 Bardach(2012)의 실용적 문제 한정이 추가되는 고유한 구조를 갖는다. 아래는 각 문단의 미시적 문장 기능과 정형구를 정밀하게 해설한다.

**문단 1 (P1). 정책 문제의 거시적 맥락 제기 (Policy Context Opening)**

첫 번째 문단은 정책 문제가 발생하는 사회적·경제적·제도적 환경을 거시적으로 제시하여, 독자가 이 문제의 구조적 배경을 이해하도록 하는 기능을 수행한다. Bardach(2012)는 좋은 문제 정의는 가능한 한 정량적이어야 한다고 강조한다.

|:--:|:--:|:--:|
| **문장 위치** | **서술 기능** | **핵심 정형구·표현** |
| S1: 거시 환경 변화 제시문 | 정책 문제가 발생하는 사회·경제·기술 환경의 변화를 사실 명제로 개시 | \[사회적 변화\]는 \[정책 영역\]의 기존 제도적 틀에 근본적 도전을 제기하고 있다 |
| S2: 문제 발현 양상 기술문 | 거시 변화가 구체적으로 어떤 정책 문제로 발현되는지 기술 | 이러한 변화는 구체적으로 \[문제 현상\]의 형태로 나타나고 있다 |
| S3: 피해 규모 수치 구체화문 | 문제의 규모·추이를 공식 통계로 구체화 | \[기관\](연도)에 따르면 \[지표\]는 \[수치\]에 달하며, 이는 전년 대비 \[N\]% 증가한 수준이다 |
| S4: 정책적 긴급성 전망문 | 이 문제가 해결되지 않을 경우의 사회적·경제적 파급 효과 전망 | 이 추세가 지속될 경우 \[연도\]까지 \[파급 효과\]가 예상된다 |
| S5: 정책 분석 필요성 예고문 | 정책적 분석과 개입의 필요성을 선언하고 다음 문단으로 전환 | 이러한 상황은 \[정책 영역\]에 대한 체계적 정책 분석의 시급성을 집약적으로 보여준다 |

**Stone(2012) 정책 역설의 핵심 개념과 미시적 서술 전략**

Stone(2012)의 Policy Paradox는 정책 분석이 \'객관적 기술\'이 아니라 \'전략적 서사 구성\'임을 논증한다. 정책 분석 논문의 미시적 구조에서 Stone의 핵심 개념들은 다음과 같이 활용된다.

<colgroup>

<tr>
<td><p><strong>Stone의 정책 분석 핵심 프레임워크</strong></p>
<p><strong>1. 형평성(Equity)의 다차원성</strong>: 분배의 공정성을 판단하는 기준이 다양함—기회의 평등, 결과의 평등, 과정의 공정성. 정책 분석에서 어떤 형평성 개념을 채택하느냐가 대안 평가 결과를 근본적으로 변화시킨다.</p>
<p><strong>2. 효율성(Efficiency)의 정치성</strong>: 비용-편익 분석은 중립적 도구처럼 보이지만, 무엇을 비용으로, 무엇을 편익으로 포함할지의 선택 자체가 가치 판단이다.</p>
<p><strong>3. 자유(Liberty)와 안전(Security)의 상충</strong>: 정책은 개인의 자유를 제한하면서 공공 안전을 확보하거나, 반대 방향을 택하는 근본적 긴장을 내포한다.</p>
<p><strong>4. 인과 이야기(Causal Stories)의 구성적 본질</strong>: 정책 문제의 원인은 '발견'되는 것이 아니라 '구성'된다. 어떤 인과 이야기가 채택되느냐가 누가 책임지고 어떤 대안이 정당화되는가를 결정한다.</p>
<p><strong>5. 수치와 측정의 정치(Numbers &amp; Counting)</strong>: 통계는 중립적 사실이 아니라 정치적 구성물이다. 무엇을 세고, 어떻게 분류하며, 어디에서 경계를 긋는가의 모든 선택이 정책적 함의를 갖는다.</p>
<p><strong>6. 상징과 은유(Symbols &amp; Metaphors)</strong>: 정책 논쟁에서 사용되는 은유('안전망', '시장의 보이지 않는 손', '사회적 투자')가 문제의 프레이밍을 결정한다.</p>
<p><em>출처: Stone (2012), Policy Paradox, 3rd ed., Part II-III</em></p></td>
</tr>

**제6장 미시 구조: 대안 평가 / 비교 분석**

**1. 평가 행렬(Evaluation Matrix) 구성 원리**

정책 대안 평가의 핵심 도구는 평가 행렬(evaluation matrix)이다. Bardach(2012)는 행렬의 행(rows)에 정책 대안을, 열(columns)에 평가 기준을 배치하고, 각 셀에서 해당 대안이 해당 기준에서 어떤 성과를 보이는지를 체계적으로 평가할 것을 제안한다.

|  |  |  |  |  |  |  |
|:--:|:--:|:--:|:--:|:--:|:--:|:--:|
| **평가 기준 →
대안 ↓** | **효과성** | **효율성** | **형평성** | **정치적 실현가능성** | **행정적 실현가능성** | **종합 평가** |
| 대안 1: 현상 유지 | 낮음 | 중간 | 낮음 | 높음 | 높음 | 비교 기준 |
| 대안 2: 규제 강화 | 높음 | 중간 | 중간 | 낮음 | 중간 | 효과 우위 |
| 대안 3: 인센티브 도입 | 중간 | 높음 | 중간 | 높음 | 높음 | 실현성 우위 |
| 대안 4: 제도 개편 | 높음 | 중간 | 높음 | 낮음 | 낮음 | 장기 최적 |

**2. 대안 평가 문단의 미시 서술 구조 (각 대안별)**

|  |  |  |
|:--:|:--:|:--:|
| **문장 위치** | **서술 기능** | **핵심 정형구·표현** |
| S1: 대안 개요 재진술문 | 해당 대안의 핵심 내용과 정책 논리(Theory of Change) 간결 재진술 | \[대안 N\]은 \[개입 수단\]을 통해 \[인과 경로\]로 \[기대 결과\]를 달성하려는 전략이다 |
| S2: 효과성 평가문 | 정책 목표 달성 가능성을 실증적 근거로 평가 | 효과성 측면에서, \[근거/선행 사례\]에 따르면 이 대안의 목표 달성 가능성은 \[높음/중간/낮음\]으로 평가된다 |

| S4: 형평성 평가문 | 정책 혜택과 부담의 분배적 공정성 분석 | 형평성 측면에서, 이 대안은 \[집단A\]에 \[혜택\]을 제공하나 \[집단B\]에 \[부담\]을 전가할 수 있다 |
| S5: 실현 가능성 평가문 | 정치적·행정적 실현 가능성 종합 평가 | 실현 가능성 측면에서, \[이해관계자\]의 \[지지/반대\] 구조와 \[행정 역량\]을 고려할 때 \[높음/중간/낮음\]으로 평가된다 |
| S6: 종합 평가 수렴문 | 기준별 평가 종합 및 해당 대안의 상대적 위치 선언 | 종합하면, \[대안 N\]은 \[기준A\]에서 우위를 보이나 \[기준B\]에서 한계를 가져, \[상대적 위치\]를 점한다 |
