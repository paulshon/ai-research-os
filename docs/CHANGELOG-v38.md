## 2. 나눔고딕 폰트 강제 적용 (첨부 폰트)

- 업로드된 NanumGothic 계열(.otf/.ttf, Light/Regular/Bold/ExtraBold)을
  `public/fonts/`에 덮어쓰기.
- 랜딩의 모든 헤드라인을 `font-sans`(시스템 폰트) → **`font-nanum-gothic`** 으로 교체하여
  첨부 폰트가 무조건 적용되도록 통일. (`@font-face`는 globals.css에 기존 등록, CSS 번들 확인)

## 3. 첨부 그림 1 지시 반영 (히어로)

- **그래디언트 메시 글래스 오브젝트 삭제**: 히어로 상단의 conic-gradient 글래스 도형 제거.
- **헤드라인 글자 크기 축소**: `clamp(2.6rem,7vw,5.2rem)` → `clamp(1.9rem,4.6vw,3.4rem)`,
  줄간격도 1.02 → 1.12로 완화.

## 4. 첨부 그림 2 지시 반영 (5개 메뉴 흐름 = 아코디언)

DesignMe Services 섹션처럼 5개 메뉴 흐름을 **아코디언**으로 재구성:
- 각 항목은 "큰 번호(01~05) + 아이콘 + 제목 + +/− 토글"의 행으로 표시.
- 클릭하면 해당 항목만 펼쳐지고(다른 항목은 접힘), +/− 아이콘이 전환(가로선 +
  세로선 회전/페이드).
- 펼침 영역에는 좌측 데모 프레임(다크 디바이스 목업 + 라이브 데모)과
  우측 단계 설명(캡션을 화살표 기준으로 번호 단계로 분해)을 함께 표시.
- 기존 좌측 탭 리스트 + 우측 단일 프레임 + 7초 자동 순환 구조를 대체.
- 5개 메뉴(문헌연구/논문구조엔진/논문크리틱/참고문헌정리/문장라이브러리)의 실제 메커니즘
  한글 데모(v36)는 그대로 유지.
