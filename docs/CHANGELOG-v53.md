# Changelog — v53

## 혼합 질적내용분석(QCA) 샘플 교체 및 브랜드어 제거

연구방법 엔진의 "혼합 질적내용분석"(Codebook-Driven QCA System v3 이식) 작업공간에서,
내장 샘플을 교체하고 기존 특정 행사명(브랜드어)을 코드 전반에서 완전히 제거했습니다. 엔진 로직(정제·빈도·코드북·
코딩·범주·주제·네트워크·해석·내보내기)과 10단계 작업공간 구조, 기본 코드북 + v3
다중시트 Excel 코드북 가져오기, xlsx/md/csv/json 내보내기는 모두 그대로 유지됩니다.

### 변경
- 내장 36문장 샘플(기존 디자인 행사 코퍼스)을 **새로운 36문장 샘플**로 교체.
  - 새 주제: **생성형 AI의 사회적 수용 담론** (영문 36문장, 100% 오프라인 데모).
  - 샘플 프로젝트명: "생성형 AI 사회적 수용 담론분석".
- 코드 전반에서 기존 행사명(국문·로마자 표기)을 **완전히 삭제**.
  - 기본 코드북명을 행사명 표기에서 "기본 내용분석 코드북"으로 변경.
  - 다중시트 가져오기 코드북명을 "가져온 코드북 (다중시트)"로 변경.
  - 작업공간 UI 문구·플레이스홀더·샘플 라벨에서 행사명 제거.
- 새 샘플에 맞춰 **기본 코드북 6코드/3상위범주를 재정의**(데모가 의미 있게 작동하도록):
  - 기술 역량 담론: 자동화·생산성 향상 / 인간-AI 협업
  - 윤리·신뢰 담론: 편향과 공정성 / 투명성과 책임
  - 사회·제도 담론: 노동시장 변화 / 교육·규제 대응
- 직전 작업에서 미완성이던 APA 관련 임시 파일(lib/apa/*)은 v53 범위 밖이라 제거.

샘플·코드북**으로 실행해 산출한 수치와, 브라우저와 동일 경로로 동작하는 TS 엔진의
출력을 **27개 항목 전부 일치**로 확인했습니다.

- 자동코딩 49배정 / 1모호. 상위범주 19 / 15 / 15.

- 커뮤니티 9개 · 모듈성 0.6149 · betweenness ai(0.078) 등 (networkx와 동일).
