## 1. 삽입한 그림/표 — 크기 조절 · 정렬(문단/줄 배치)

`apps/web/components/editor/rich-text-editor.tsx` 에 미디어 편집 기능을 추가했습니다.

- 본문에 삽입된 **그림(img)·표(table)를 클릭하면 미디어 툴바**가 위에 떠오릅니다.
  - **크기**: 작게(40%) / 보통(70%) / 크게(100%) 프리셋.
  - **이미지 자유 리사이즈**: 우하단 코너 핸들을 드래그하여 8~100% 범위로 조절.
  - **정렬**: 왼쪽 / 가운데 / 오른쪽 (왼쪽·오른쪽은 본문이 옆으로 흐르는 float 배치).
  - **줄/문단 전환**: 인라인(줄 안 배치) ↔ 블록(문단 단독 배치) 토글.
  - **삭제**.
- 선택된 미디어는 파란 외곽선으로 강조 표시. 변경은 즉시 본문 HTML에 반영되어 저장/내보내기에 포함.
- 삽입 미디어 기본 스타일(반응형 max-width, 라운드, 가운데 정렬) 적용.

## 2. 엑셀에서 만든 그래프 가져오기 (차트 재현)

SheetJS는 엑셀에 **내장된 차트(DrawingML)** 를 읽지 못합니다. 그래서 표 데이터를 그대로
가져오던 기존 동작에 더해, **시트의 숫자 데이터로 동일한 콤보 차트를 재현**하여 함께 삽입합니다.

- `apps/web/app/(dashboard)/writing/page.tsx` 에 `buildChartFromRows()` 추가.
  - 첫 행을 라벨로, 이후 숫자 행들을 계열로 인식.
  - 계열1은 **막대(파랑)**, 계열2는 **선(주황, 보조축 · % 자동 인식)** 으로 그리는 Excel 스타일 콤보 차트.
  - HTML5 Canvas로 PNG를 생성(추가 라이브러리 없음)하여 `<img>`로 삽입.
- CSV/XLSX 가져오기 시: **표 + 차트**가 함께 삽입됩니다(숫자 데이터가 있을 때).
- 삽입된 차트 이미지는 1번의 미디어 툴바로 크기·정렬을 동일하게 조절할 수 있습니다.
- 첨부 그림(예: a/b/c/d/f = 10/50/30/20/5 + 백분율)의 막대+선 그래프가 그대로 재현됩니다.
