# CHANGELOG v29 — 미드나잇 색상 변경 · 사이드바 하단 패널 테마 연동

## 1. 미드나잇 테마 색상 변경 (블루-블랙)
첨부 이미지(4번)의 색상에 맞춰 **미드나잇** 테마를 기존 퍼플-블랙(`#0d0212`)에서
**블루-블랙**(`#01030c`)으로 변경했습니다. 거의 검정에 가깝되 미세한 푸른 기운이
도는 톤으로, 순수 블랙 테마와 구분됩니다.

| 토큰 | 이전(퍼플) | 변경(블루-블랙) |
|------|-----------|----------------|
| `--bg` | `#0d0212` | `#01030c` |
| `--bg2`(사이드바) | `#160a1e` | `#070b18` |
| `--bg3` | `#1f1228` | `#0e1426` |
| `--bg4` | `#2a1a34` | `#161d33` |
| `--surface` | `#1c1026` | `#0b1020` |
| `--surface-deep` | `#08000c` | `#000208` |

- `app/globals.css`의 `:root` 기본값과 `[data-theme="midnight"]` 모두 갱신
- `components/theme/theme-switcher.tsx`의 미드나잇 스와치도 `#01030c`로 갱신

## 2. 사이드바 하단 "프로젝트 워크스페이스" 패널 테마 연동
스크린샷에서 지적된 문제: 테마를 바꿔도 사이드바 **좌측 하단의 프로젝트 워크스페이스
박스**(새 프로젝트 / 초기화 / 저장·임시저장·불러오기)가 사이드바와 다른 색으로 남아
있었습니다.

원인: 해당 패널(`ProjectSavePanel`)의 컨테이너 배경이 `bg-[#0f1118]`(→`--surface-deep`,
더 어두운 톤)이라 사이드바(`bg-[#13161e]`→`--bg2`)와 달랐습니다.

→ 패널 컨테이너 배경을 사이드바와 **동일한 `bg-[#13161e]`(→`--bg2`)** 로 변경.
   이제 세 테마(미드나잇/차콜/블랙) 모두에서 하단 패널이 사이드바와 **똑같은 색**으로
   함께 전환됩니다.
- `components/save/project-save-panel.tsx`
