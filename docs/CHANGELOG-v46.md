## 스플래시 스크린 강화 (PulseGlow · 5초)

`apps/web/components/marketing/splash-screen.tsx` 를 갱신했습니다.

### 1) 재생 시간 5초
- 전체 표시 시간 3초 → **5초**(`SPLASH_MS = 5000`). 이후 페이드/스케일 아웃으로 랜딩 노출.
- 텍스트 아래 **로딩 진행 바**가 약 4.2초에 걸쳐 0→100% 채워져 5초 흐름을 시각화.
- 타임라인: 등장(0~0.8s) → 박동/호흡(0.8~4.2s) → 분해·퇴장(4.2~5.0s).

### 2) 로고 PulseGlow (Uplift UI "Pulse Glow" 참고)
첨부 로고(보라 그라데이션 라운드 + 사방별)에 그대로 글로우를 입혔습니다.
- **central glow**: 로고 뒤 보라/블루 방사형 글로우가 심장박동처럼 크기·밝기를 반복 진동
  (scale 0.92↔1.12, opacity 0.45↔0.85, 1.6s 루프).
- **ring 확산**: 동심원 링 3개가 시작 시점을 0.9s씩 어긋나게 하여 바깥으로 퍼지며 페이드
  (scale 0.85→2.1, opacity 0.55→0, 2.7s 루프) — 끊임없는 박동 ripple.
- 로고 박스 자체도 `boxShadow`가 글로우 강도에 맞춰 박동(28px↔55px 반경)하고, 내부 사방별이
  미세하게 회전·스케일.

### 3) 전 기기 적용 (데스크탑/태블릿/모바일)
- 랜딩 "/"의 첫 화면 오버레이로 표시되며 모든 기기에서 동일하게 재생.
- 모바일은 세로(Column) 배치, 태블릿/데스크탑은 가로(Row) 배치 + 글로우/링 반경 확대.
- 정적 프리렌더 HTML 첫 페인트에 스플래시 오버레이가 포함되어 로드 즉시 표시.

### 접근성
- `useReducedMotion` 감지 시 확산 링/회전 루프를 끄고 정적 글로우만 유지, 재생 시간 2.2초로 단축.
- `role="img"` + `aria-label="AI Research OS 로딩"`.
