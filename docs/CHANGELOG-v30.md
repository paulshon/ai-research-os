# CHANGELOG v30 — 모바일 UX 개선 (메뉴 통일 · 슬라이드 시트 · 페이지 디자인 · 크리틱 실행 수정)

첨부 스크린샷 5건의 지적사항을 모두 반영했습니다.

## 1. 하단 메뉴 명칭 통일 + 슬라이드업 시트 (Image 1, 4)
- 사이드바는 "AI Tools", 하단 모바일 메뉴는 "엔진(AI Engines)"으로 불일치하던 것을
  **"AI Tools"로 통일**.
- 하단의 AI Tools 확장 메뉴를 기존의 좁은 인라인 패널에서 **하단 슬라이드업 시트**로
  재설계: 백드롭 + 그랩 핸들 + 3열 그리드, 부드러운 slide-up 애니메이션.
- 항목 탭 시 시트가 자동으로 닫힘.
- 파일: `components/dashboard/dashboard-shell.tsx`, `app/globals.css`(slide-up/fade-in 추가)

## 2. 하단 슬라이드 메뉴에서 녹색 "참고문헌" 버튼 제거 (Image 4)
- 참고문헌 패널 버튼(CitationButton)을 하단 시트에서 **삭제**.
- 참고문헌 관련 진입점은 상단바 우측에 유지: **참고문헌 정리(/references) 링크** +
  **참고문헌 패널 버튼** — 모든 페이지(논문 작성·참고문헌 정리 포함) 상단 오른쪽에 노출.

## 3. 논문 작성(집필) 페이지 모바일 UI 정리 (Image 2)
- 상단바에 "섹션을 선택하세요"가 두 번 중복 표시되고 AI생성/Copilot/복사/저장
  버튼이 좁은 한 줄에 뭉개지던 문제 수정:
  · 데스크탑은 현재 섹션명 + 액션, 모바일은 타이틀 + 아이콘 액션으로 분리
  · 복사 버튼에 실제 복사 동작 연결(에디터 텍스트 → 클립보드)
  · 저장 버튼 컴팩트화
- 모바일 챕터/소절 선택 드롭다운을 둥근 카드 + 셰브론 아이콘으로 재디자인,
  폰트·여백 정리.
- 파일: `app/(dashboard)/writing/page.tsx`

## 4. AI 멘토링 페이지 모바일 선택기 재디자인 (Image 3)
- "학술적 절차 검증멘토링 — 양적연구 전문"이 그냥 기본 `<select>`라 붙여놓은 듯
  어색하던 것을, **멘토 아바타 + 이름 + 전문분야 + 특성 태그**가 보이는 스타일
  카드형 드롭다운으로 재설계.
- 피드백 버튼(서론/방법론/결과/전체)이 모바일에서 줄바꿈되도록 수정.
- 파일: `app/(dashboard)/advisor/page.tsx`

## 5. 논문 크리틱 페이지 모바일 디자인 + 실행 버그 수정 (Image 5)
- 뷰어 툴바(원문 뷰어 / 텍스트입력 / AI분석 / 필터 / 폰트 A-/A+)가 좁은 한 줄에
  넘쳐 깨지던 것을 **모바일에서 wrap되는 반응형 레이아웃**으로 재구성, 아이콘·셰브론
  정리.
- 모바일 3-pane 전환 탭 아이콘을 lucide 아웃라인으로 교체.
- **실행 버그 수정**: 논문 업로드 후 AI 분석이 실행 안 되던 문제.
  · 분석 텍스트 소스를 우선순위로 수집(PDF 페이지 텍스트 → UODE 블록 → docText)하도록
    `aiAnalyze` 보강. 기존엔 `isPdf`면 무조건 페이지 텍스트만 봐서, 텍스트가 비면
    조용히 중단됐음.
  · 분석할 텍스트가 없을 때(스캔 이미지 PDF 등) **명확한 안내 알림** 표시
    (`critique.noTextAlert`, ko/en/zh).
  · 분석 결과가 생성되면 모바일에서 **크리틱 패널로 자동 전환**되어 결과가 바로 보이게 함.
- 파일: `app/(dashboard)/critique/page.tsx`, `lib/i18n/locales/*-pages.ts`
