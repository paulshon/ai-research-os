# CHANGELOG — v51

논문구조엔진(Structure Engine)에 **방법론 개발형·비교/역사형의 7개 구조엔진 유형을
실제 챕터 구조로 구현**한 릴리스입니다. 메뉴에는 노출되어 있었으나 챕터 데이터가 없어
선택 시 ⓵ 양적 연구의 기본 구조로 폴백되던 7개 유형(⑱~㉔)을, 첨부된 각 유형별
『거시·미시구조 완전 가이드』를 정밀 분석하여 기존 17개 유형과 동일한 정본 챕터
구조(거시구조 + 미시구조 + Good/Bad 패턴)로 채웠습니다.

---

## 1. 신규 구현된 7개 구조엔진 유형

각 유형의 챕터 구조는 해당 가이드의 거시적 기능(장별 기능)·미시적 문단 구조·좋은
문장 패턴·탈락 패턴을 `Chapter` 모델(`macro`/`micro`/`goodPatterns`/`badPatterns`)로
이식했습니다.

| 코드 | 유형 | 장 수 | 거시·미시 전거 |
|------|------|:----:|----------------|
| `sv`   | ⑱ 척도 타당화 | 9 | Kline(2016) SEM 6단계 · Brown(2015) CFA 3단 평가·Table 4.7 |
| `algo` | ⑲ 알고리즘/모델 | 9 | Goodfellow et al.(2016) 실용 설계 4단계 · CLRS(2009) 의사코드·루프 불변식·점근표기 |
| `fw`   | ⑳ 프레임워크 | 9 | Miles, Huberman & Saldaña(2014) 구성요소·관계 · Anfara & Mertz(2015) 개념→구성→명제→이론 |
| `comp` | ㉑ 비교 연구 | 7 | Ragin(1987/2014) 비교 전략·결합적 인과성 · George & Bennett(2005) SFC·과정추적 |
| `hist` | ㉒ 역사 연구 | 6 | Tosh(2010) 기술·서사·분석·다층 인과 · Storey(2013) 문단 3요소·사료→추론→논증 |
| `da`   | ㉓ 담론 분석 | 8 | Fairclough(2003) 3차원 분석 틀 · Van Dijk(2008) 이념적 사각형·사회-인지 모델 |
| `bio`  | ㉔ 전기적 연구 | 6 | Caine et al.(2022) 3차원 탐구공간·함께 살아가기 · Polkinghorne(1988) 엮어내기 |

### 각 유형의 거시 구조(장 배열)

- **⑱ 척도 타당화**: 서론 → 구성개념 정의 → 문항 개발·내용타당도 → 연구방법 → EFA(연구1)
  → **CFA(연구2)★** → 타당도·신뢰도 증거 → 논의 → 결론
- **⑲ 알고리즘/모델**: 서론 → 관련 연구 → 문제 정식화 → **제안 방법★** → **이론적 분석★**
  → 실험 설계 → 결과·절제 분석 → 논의·한계 → 결론
- **⑳ 프레임워크**: 서론 → 기존 이론적 지형 → 개발 방법 → **구성요소★** → **관계·명제★**
  → 통합 프레임워크 → 적용·예시 → 논의 → 결론
- **㉑ 비교 연구**: 서론 → 이론적 배경 → 연구 설계 → 사례 기술 → 비교 분석 → 논의 → 결론
- **㉒ 역사 연구**: 서론(논제) → 연구사 검토 → 사료와 방법 → 본론(기술·서사·분석)
  → 인과와 해석 → 결론
- **㉓ 담론 분석**: 서론 → 이론적 배경 → 연구 설계 → 텍스트 분석 → 담론 실천 분석
  → 사회 실천 분석 → 논의 → 결론
- **㉔ 전기적 연구**: 서론(연구 퍼즐) → 철학적 토대 → 3차원 탐구공간 → 함께 살아가기
  → 현장→연구텍스트 → 결론

## 2. 변경 사항

- `apps/web/lib/research-data.ts`
  - `CHAPTERS_SV`, `CHAPTERS_ALGO`, `CHAPTERS_FW`, `CHAPTERS_COMP`, `CHAPTERS_HIST`,
    `CHAPTERS_DA`, `CHAPTERS_BIO` 7개 정본 챕터 배열 신규 정의(총 51개 장).
  - `getChapters()` 매핑 테이블에 `sv/algo/fw/comp/hist/da/bio` 7개 항목 추가.
    → 더 이상 기본값(`CHAPTERS_QUANT`)으로 폴백되지 않음.
- i18n: 7개 유형의 메뉴 라벨·카테고리 키(`research.typeSv` … `research.typeBio`,
  `research.catMethods`/`research.catComparative`)는 ko/en/zh에 이미 존재하여 그대로 동작.
  한국어는 작성된 정본 챕터 제목을 그대로 사용하며, 정규 섹션(서론·이론적 배경·연구방법·
  논의·결론)은 기존 `KO_TITLE_TO_KEY` 규칙으로 다국어 매핑됨.
- `docs/논문구조엔진파일/`: 7개 유형의 원본 가이드(.docx)를 기존 가이드와 함께 보존.
