# CHANGELOG — v42

첨부된 피드백 문서와 8장의 화면 주석(빨간색 지시), Aivora 사이드바 참조 디자인을 반영한
**UI/UX 전면 정비** 릴리스입니다. 기능(IA)은 v41을 유지하고, "정보 밀도·가독성·공간 활용"을
끌어올리는 데 집중했습니다.

> 참고: 데스크탑/태블릿/모바일은 별도 빌드가 아니라 하나의 Next.js 웹앱의 반응형
> 레이아웃이며, 아래 변경은 사이드바·상단바·각 페이지에 공통 반영됩니다.

---

## 1. 전체 타이포그래피 +3px (공통)

모든 화면 상단에 반복된 "전체적으로 글자를 3포인트 크게" 지시를 반영하여, 대시보드 영역의
모든 텍스트 토큰(`text-[Npx]`)을 일괄 +3px 확대했습니다. 본문 13–14 → 16–17px, 메뉴 13 → 16px,
카드 제목 16 → 19px, 섹션 제목 20 → 23px, 페이지 타이틀 28 → 31px 수준으로 상향됩니다.
(대상: 대시보드 전 페이지 + 문헌 컴포넌트 24개 파일)

## 2. 중앙 콘텐츠 확장 · 좌우 여백 축소 (1순위)

가장 큰 문제로 지적된 "중앙 콘텐츠가 좁고 좌우 여백이 과도함"을 해결했습니다.

- 페이지 래퍼 최대폭을 `max-w-5xl/6xl/4xl`(1024–1152px) → **`max-w-[1680px]`** 로 확대.
- 외곽 패딩(`lg:p-8`/`md:p-8`)을 축소하여 좌우 거터를 줄임.
- 결과적으로 좁은 콘텐츠 대신 화면을 가득 채우는 와이드 레이아웃으로 전환.

## 3. 좌측 사이드바 전면 재설계 (Aivora 참조)

- 폭 224 → **256px**, 메뉴 글자 12.5 → **15px**(weight 500), 아이콘 14 → **18px**, 행 높이 확대.
- 활성 항목에 **좌측 강조 바 + 틴트 배경** 적용으로 존재감 강화.
- 로고 헤더 확대(그라데이션 로고 + 설정 어포던스), 섹션 라벨(Research Flow / AI Tools) 가독성 향상.
- 모바일 사이드바 오버레이 폭(240 → 264px)도 함께 보정.

## 4. 상단 프로세스 메뉴 강조

- 높이 40 → **52px**, 글자 11 → **15px / weight 600**, 패딩 확대.
- 선택 상태: **글로우 언더라인(box-shadow) + 배경 하이라이트** 로 변경(기존 단순 배경 → 시인성 강화).

## 5. 하단 워크스페이스/프로필 영역 재설계 (image 8 "가장 엉망")

좌측 하단의 프로젝트 워크스페이스 컨트롤을 컴팩트하고 세련된 카드형으로 전면 개편했습니다.

- 프로젝트명 카드(폴더 아이콘 + 인라인 편집), 연결 `.aros` 파일 상태 표시(점 인디케이터).
- 주 동작(저장/불러오기)과 보조 동작(임시저장/초기화)을 시각적으로 분리, 버튼 9px → 12–13px 확대.
- 사용자 프로필 행을 카드형으로 확대(아바타 + 이름/이메일).

## 6. 검토·검증 — 우측 빈 공간을 라이브 품질 레일로 (image 4)

1열 → **2열 구조**로 변경하고 우측에 sticky 컨텍스트 레일을 추가했습니다.

- 종합 품질 등급(도넛 게이지) + 종합 점수.
- 표절 유사도 / 맞춤법 정확도 / AI 생성률 3지표 바.
- 페이지별 위험도 미니 차트, AI 진단 요약(표절·맞춤법·AI 의심 건수).
- 모든 수치는 기존 Sentence Map 엔진 결과를 실시간으로 반영.

## 7. 논문일정 — 우측 컨텍스트 패널 (image 5)

비어 있던 우측 영역에 일정 요약 패널을 추가했습니다.

- 전체 진행률 도넛 + 완료 태스크/제출 준비도, 단계별 진행 바.
- "다음 할 일"(미완료 태스크 상위, 클릭 시 토글), AI 추천 카드.

## 8. 문헌연구 — 검색 하단 분석 대시보드 (image 2)

검색창 아래 거대한 공백을 분석 대시보드로 채웠습니다.

- 메트릭 4종(수집 논문·연도 범위·고유 저널·고유 저자).
- 연도별 분포 막대 차트 + 주요 저널 칩, 상세(네트워크·군집) 분석 링크.
- 검색 결과가 없을 때도 의도된 플레이스홀더로 보이도록 설계.

## 9. 대시보드 정보 밀도 향상 (image 1)

- 콘텐츠 폭 1000 → 1680px, 헤더 라벨 v21 → v42.
- 빠른 시작 3 → **4 컬러 타일**(이어 쓰기 / 검증 실행 / 문헌 갭 탐색 / 논문일정).

## 10. 논문작성 — 본문 확대 + Copilot 접기 (image 3)

- 중앙 에디터 본문 폭 680 → **1040px**(좌우 공백 제거, 외곽 패딩 축소).
